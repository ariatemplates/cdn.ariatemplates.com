Usage

To use this CDN in your Aria Templates project, include the following line in your bootstrap file:

<script src="http://cdn.ariatemplates.com/at<version>.js[?params]"></script>

Where:

version is the version label of the framework, using either dots or dashes, or latest to get the latest build available.

params contains optional parameters such as skin to request the Aria skin or dev to fetch unminified files.

Examples

<script src="http://cdn.ariatemplates.com/at1.4.9.js"></script>
<script src="http://cdn.ariatemplates.com/atlatest.js?skin"></script>
<script src="http://cdn.ariatemplates.com/at1.4-2.js?dev&skin"></script>
